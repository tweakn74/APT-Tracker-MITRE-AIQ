📋 How to Build Your FIN7 Assessment in AttackIQ

  • Step 1: Login to AttackIQ platform and navigate to "Assessments"
  • Step 2: Click "Create Custom Assessment" (don't use the FIN7 template - build your own!)
  • Step 3: For each technique in the purple AttackIQ row above, search by T-Code (e.g., T1003.001)
  • Step 4: Select the scenario variant that best matches FIN7's methodology
  • Step 5: Add all 12 scenarios to create a complete FIN7 emulation assessment
  • Step 6: Run the assessment against your environment to validate security controls
💡 Pro Tip: The T-Codes in the AttackIQ row are your search keys. Each T-Code will return multiple scenario options in AttackIQ - pick the ones using tools like Mimikatz, PowerShell, PsExec, and RDP that FIN7 actually uses. This custom approach teaches you threat modeling better than using pre-built templates!

🔍 How to Search These Sites for Specific APT Groups

📊 MITRE ATT&CK - Finding Groups by Sector & Tactics

Example: Find APT groups targeting finance/banking + using ransomware + extortion

  • Go to attack.mitre.org/groups/
  • Use browser search (Ctrl+F) for keywords: "financial", "finance", "banking"
  • Click group profiles, check "Techniques Used" for T1486 (ransomware) or T1499 (extortion)
  • Check group descriptions for "ransom" or "extortion" keywords
Example Groups: FIN7 (G0046), FIN6 (G0037), Carbanak (G0008), Wizard Spider (G0102)

🎨 MITRE ATT&CK Navigator - Visual Technique Mapping

Find groups using specific TTPs:

  • Open ATT&CK Navigator
  • Click "Create New Layer" → "Create Layer from ATT&CK Data"
  • Select techniques: T1486 (Data Encrypted for Impact), T1041 (Exfiltration), T1003 (Credential Dumping)
  • The Navigator shows which groups use these techniques together
  • Export the layer or screenshot for presentations
Pro Tip: Create a custom layer with all techniques from your environment's critical assets, then see which APT groups match that pattern!

🔒 Ransomware.live - Track Active Ransomware Operations

Find ransomware groups actively targeting specific sectors:

  • Visit ransomware.live
  • Check the "Recent Attacks" feed for victim industry types
  • Use the group profiles to see which gangs target finance/banking
  • Click on specific groups (LockBit, BlackCat, ALPHV, etc.) to see their victim lists
  • Filter by country or industry if available
Example: Click "LockBit" → See all recent victims → Filter for finance sector

🏛️ CISA Alerts - Government Advisories Discovery

Search for official threat actor advisories:

  • Go to CISA Cybersecurity Advisories
  • Use search box: Try "ransomware financial" or "APT banking"
  • Filter by date range to get recent threats
  • Look for alerts with format "AA##-###A" for detailed APT/ransomware advisories
  • These include IOCs (Indicators of Compromise) and YARA rules you can download FREE
Example: Search "Volt Typhoon" or "FIN7" to get official detection signatures

⚔️ AttackIQ Platform - Find Emulation Scenarios

Search AttackIQ for testable scenarios:

  • Login to AttackIQ platform
  • Go to "Assessments" → "Browse Library"
  • Search by APT group name: "FIN7", "Lazarus", "APT29"
  • OR search by technique: "T1003" (credential dumping), "T1486" (ransomware)
  • OR search by sector: "financial services", "healthcare", "energy"
  • Combine searches: "ransomware + financial" to find relevant scenarios
Platform Search: "FIN7" → Returns pre-built assessment with 17+ scenarios
Custom Build: Search T1566.001 + T1003.001 + T1486 individually to build your own

🔍 Vendor Blog Research (Mandiant, CrowdStrike, Unit 42, SentinelLabs)

Search vendor blogs for detailed APT analysis:

  • Mandiant: Search blog for "FIN7 finance" or "ransomware banking"
  • CrowdStrike: Use their Adversary Universe or blog search
  • Unit 42: Search reports by industry vertical
  • SentinelLabs: Filter research by threat actor or malware family
Search Strategy: Combine "APT name" + "sector" + "year" to find the most recent intelligence reports. Example: "FIN7 financial services 2024"

🎯 Complete Workflow Example: Finance-Targeting Ransomware Groups

Step-by-step process to research "Finance-Targeting Ransomware Groups":

  • Step 1 - MITRE ATT&CK: Go to groups page → Search "financial" → Find FIN7, FIN6, Carbanak, Wizard Spider
  • Step 2 - Verify Ransomware: Check each group profile → Look for T1486 technique → Confirm ransomware ties
  • Step 3 - Check Active Operations: Visit Ransomware.live → Search for these groups → See recent finance sector victims
  • Step 4 - Get Government Intelligence: Search CISA for "FIN7" or "ransomware financial" → Download IOCs and detection rules
  • Step 5 - Deep Dive Research: Search Mandiant/CrowdStrike blogs for "FIN7 2024" → Get latest TTPs and tools
  • Step 6 - Build AttackIQ Assessment: Search T-codes from group profile → Select scenarios → Test your defenses
  • Step 7 - Validate & Document: Cross-reference findings across all sources → Build threat profile → Present to stakeholders
Result: You now have a complete threat profile, recent attack data, detection rules, and testable scenarios for financial sector ransomware threats!
